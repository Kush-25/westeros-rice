# Westeros Customiser

A bash script that lets you pledge allegiance to a Great House and instantly theme your KDE Plasma desktop accordingly.

Inspired by Game of Thrones / A Song of Ice and Fire.

## Features (planned)
- Interactive house selection (Stark, Lannister, Targaryen, ...)
- Apply custom:
  - Wallpapers
  - KDE color schemes
  - Icons (sigil overrides?)
  - Konsole terminal profiles
  - Optional: prompt tweaks, MOTD quotes
- Safe & reversible (backups, dry-run mode)
